* action=settask *

This module only accepts POST requests
Parameters:
  joblist      - Joblist Id
  schedule     - Scheduled Time (YYYY-mm-dd HH:MM:ss )
  source_file  - File to Encode
  notify       - Notification (Set "1" to notify User, set "2" for Group and leave empty to disable)
